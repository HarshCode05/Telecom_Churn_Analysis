## 📌 Overview

Customer churn is one of the biggest challenges in the telecom industry.
This project provides a **complete data-driven solution** to analyze customer behavior and predict churn using Machine Learning.

It combines:

* 📊 Data Analysis
* 🤖 Machine Learning
* 🌐 Interactive Dashboard

---

## 🎯 Objectives

* 📉 Identify churn patterns
* 🔍 Analyze customer behavior
* 🤖 Build a predictive model
* 📊 Create a real-time dashboard for insights

## 🤖 Machine Learning Model

* **Algorithm:** Random Forest Classifier
* **Problem Type:** Classification
* **Target Variable:** Churn (0 = No, 1 = Yes)

### 📥 Input Features:

* Age
* Gender
* City
* State
* Calls Made
* SMS Sent
* Data Usage
* Estimated Salary

### 📈 Model Performance:

```
Accuracy: 85%
```
